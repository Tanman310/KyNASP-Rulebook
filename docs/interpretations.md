# Interpretations

KyNASP Interpretations are written clarifications of how an existing KyNASP rule is to be applied. Interpretations do not create new rules, expand enforcement authority, or contradict NASP® National Tournament Rules. Where a conflict exists, NASP® National Rules control.
Interpretations are issued under the administrative authority of the KyNASP State Coordinator. This site presents interpretations as a structured publication method for clarification purposes only and does not create new rulemaking authority.

## Living Interpretation Set
Interpretations are maintained as a living set located at:
`docs/interpretations/current/`

Each interpretation exists in its own folder named by its issued date using the format:
`yyyy-mm-dd_description`

Example:
`2026-03-03_distance-learning-eligibility/`

The folder date reflects the Issued date of the interpretation. Each interpretation also includes an Effective date. “Effective” means binding as of the stated date.
Interpretations remain in effect until marked Superseded or Archived.

## Publication Requirement
Only interpretations that are published within the docs/interpretations/current/ directory and displayed on the public site are treated as official for purposes of this rulebook presentation.

Unpublished guidance, informal communications, or verbal rulings are not treated as official interpretations unless formally published.

## Inline Display
Interpretations appear inline under the affected rule sections as collapsible expandable blocks. An interpretation may apply to multiple rules. The canonical interpretation file lists all affected rule references.

## Supersession
If an interpretation is replaced:

- The original interpretation remains published.
- Its status is changed to Superseded.
- The replacing interpretation is referenced.
- Interpretation IDs are never reused.


Archival of interpretations occurs only upon declaration of a major rulebook revision.
